# NoSQLSocialNetAPI

# Description

This project is an API for a social network web application built using MongoDB, Express.js, and Mongoose. The API allows users to share thoughts, react to friendsâ€™ thoughts, and manage their friend list. It leverages NoSQL databases for handling large amounts of unstructured data typically found in social networking platforms.

# Features

- User Management: Users can be created, updated, and deleted.
- Thought Sharing: Users can create, read, update, and delete thoughts.
- Reactions: Users can react to thoughts by adding or deleting reactions.
- Friend Management: Users can add or remove friends from their friend list.

# Installation

- Ensure you have MongoDB installed on your local machine.
- Clone this repository to your local machine.
- Run npm install to install dependencies.
- Start the server by running npm start.
